## Instalacion

npm i -g @loopback/cli

- lb4 nombreapp
- lb4 datasource
- lb4 model
- lb4 repository
- lb4 controller

* Para que funcione es comando "npm start", se puede situar en la carpeta dist o en el mismo proyecto
